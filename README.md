#  E-Commerce Sales Analysis (Pandas Project)

##  Overview
This project analyzes e-commerce sales data using Python (Pandas) to extract business insights.

##  Tools Used
- Python
- Pandas

##  Dataset
- Orders
- Customers
- Products

##  Key Operations
- Data Cleaning (missing values, datetime conversion)
- Merging multiple datasets
- Feature Engineering (year, month, day)
- Filtering & conditional analysis
- Apply, Map, ApplyMap usage
- GroupBy & Ranking
- Pivot Tables (dashboard-level analysis)
- Sorting & String operations
- Transform (customer & category comparison)

##  Key Insights
- Delhi generates the highest revenue
- Electronics category dominates revenue
- March shows peak sales performance
- A few customers contribute major revenue (high concentration)
- Laptop is the primary revenue driver

##  Output
Final cleaned dataset exported as CSV
